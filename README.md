# Proyecto de Facturas en Elm

Este proyecto es una aplicación de gestión de facturas construida con **Elm**.

## Configuración de Desarrollo

El proyecto utiliza **Bun** para la gestión de dependencias y la ejecución de herramientas de desarrollo.

### Scripts disponibles

- `bun run watch`: Inicia `elm-watch` para compilar y vigilar cambios en el código Elm.
- `bun run serve`: Inicia un servidor de desarrollo (`live-server`) en `http://localhost:8080`.

### Desarrollo

Para iniciar el entorno de desarrollo, abre dos terminales y ejecuta:

1. `bun run watch`
2. `bun run serve`
